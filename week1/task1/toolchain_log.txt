# Task 1: RISC-V Toolchain Install & Sanity Check

## Step 1: Downloaded Toolchain

$ wget https://vsd-labs.sgp1.cdn.digitaloceanspaces.com/vsd-labs/riscv-toolchain-rv32imac-x86_64-ubuntu.tar.gz

## Step 2: Extracted the Toolchain

$ tar -xzvf riscv-toolchain-rv32imac-x86_64-ubuntu.tar.gz

## Step 3: Moved Toolchain to $HOME/riscv

$ mv riscv $HOME/riscv

## Step 4: Added Toolchain to PATH

$ echo 'export PATH=$HOME/riscv/bin:$PATH' >> ~/.bashrc
$ source ~/.bashrc

## Step 5: Verified Tool Versions

$ riscv32-unknown-elf-gcc --version
